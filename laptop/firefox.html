If you are having trouble using Firefox, this page should help.

Could not connect to page error.

A photo of the error in action.

First, if you haven't already, make sure you have local administrator access using the methods in the website listed below:

How to get administrator access

If you are having issues connecting to websites in Firefox, check that the Securly certificate is in Firefox. If the Securly certificate is not there, follow the steps below.

(LOCAL ADMIN REQUIRED) Press Windows key + "R" on your keyboard, and type in gpedit.msc

Then, on the sidebar on the left, navigate to Administrative Templates/System/Internet Communication Management/Internet Communication Settings

Next, double click Turn off Automatic Root Certificates Update and click "Enable", as shown below. Then. click "Apply" in the bottom right.

Then, restart your computer. The certificate should now be in Firefox, and you can continue following the Securly removal guide.

CAUTION: MAKE SURE TO VERIFY THAT THE CERTIFICATE IS DELETED AFTER EVERY SHUTDOWN OR RESTART OF THE LAPTOP.
