How to remove Securly.

First, if you haven't already, make sure you have local administrator access using the methods in the website listed below:

How to get administrator access

Next, get the Firefox browser and follow the instructions to install Hoxx VPN. First, download the extension file from the link below.

Firefox Installer Download (Run as administrator) Hoxx VPN Download

Once the extension file has downloaded, simply click "Continute to Installation"

Then, go to about:preferences and search up "Certificates".

Open the certificate menu, scroll down until you see "Securly", select it, and then click "Delete".

Securly should now be uninstalled on Firefox. Make sure to use a VPN!

CAUTION: MAKE SURE TO VERIFY THAT THE CERTIFICATE IS DELETED AFTER EVERY SHUTDOWN OR RESTART OF THE LAPTOP.
