Getting the materials

First, we must create a Windows 10 Installer USB. This can be done by getting any *blank* USB with at least 8GB of available space.

Then, once you have a blank USB ready, download the Windows 10 Media Creation tool. (link below)

Media Creation Tool download

After following the instructions in the Media Creation Tool, you should now have a USB with the Windows 10 installer on it. Eject the USB from the computer and get ready for the next step of the process.

Preparing the laptop and using the Windows 10 Installer:

Preparing the laptop.

First, shut down the computer fully and plug in the USB drive that contains the installer. After plugging in the USB, power on the laptop and repeatedly press the "F12" key until the BIOS menu appears. (pictured below)

The BIOS screen loading after pressing F12, with the "Preparing one-time boot menu" message in the top-right corner.

Once the BIOS menu is fully loaded, select the USB stick under "EUFI" boot options. (It should be the only option besides "Windows Boot Manager"

The BIOS menu fully loaded, with the USB drive selected.

After selecting the USB and pressing "ENTER" on your keyboard, you should be greeted with this screen. Select "Windows Setup (64-bit)"

After all of the files have loaded, the Windows Setup dialog will appear. Click "Next".

Then, click "Repair your computer"

After that, this screen will appear. Click "Troubleshoot", and then "Command Prompt".

Now we have gained access to an administrator command prompt. Be careful, if you do not follow the steps carefully you could completely ruin your computer.

First, type in C:, as shown below.

After that, type in CD Windows\System32. This is case-sensitive, meaning you have to type it exactly as it appears.

(With capital letters)

The final command prompt step will be to type in COPY cmd.exe sethc.exe. When it asks for confirmation, type Y.

After you are done using the command prompt, exit the program and shut down your laptop, to begin the next step of the process.

Getting the administrator account:

After shutting down your laptop, boot it up as you normally do. However, do NOT sign in yet.

When you arrive at the lock screen, repeatedly press the "SHIFT" key on the keyboard (basically spam it) until a command prompt window opens up, as seen below.

Once the prompt has opened, type net user sktech password, and exit the window once it has completed succesfully. Then, sign in with your *student* account.

After signing in, go to "Settings", click on "System", and scroll all the way down on the left hand side until you find "About". Once you are in the "About" menu, take note of the "Device Name". Take a photo of it, write it down, or do whatever is necessary to remember it. This is very important.

Alright, now all of the difficult stuff is done, and we can begin actually using the administrator account. Whenever Windows needs administrator privileges to do something, it opens up something called the "UAC" panel, which looks something like this:

Now, instead of entering your information, you are going to click on "More Choices", and then scroll down to "Use a different account".

Now comes the part where the device name comes in handy. For the username, type in your device name followed by \sktech.

The password is simply just password.

For instance, if your device name is "2022-XXXXXXX", then you would type in 2022-XXXXXXX\sktech as the username.

Update: Alternatively, you can just use .\sktech as the username.

Well, that's all there is to it. If you ever need administrator access, just make sure to always sign in with the "sktech" account.
